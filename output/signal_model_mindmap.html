# 卖出因子 类型 评分规则 分值 说明
1 止盈因子 止盈 盈利≥20% +40
盈利≥10% +25
盈利≥5% +10 		0 ~ +40 达到止盈线时触发，盈利越多卖出信号越强
2 止损因子 止损 亏损≥10% +45
亏损≥5% +30
亏损≥3% +15 		0 ~ +45 触发止损线，亏损越深卖出信号越强，保护本金
3 人气衰退 趋势 5分钟排名下降≥15位 +20
5分钟排名下降≥8位 +10
1小时排名下降≥20位 +15 		0 ~ +20 人气榜排名快速下滑，市场关注度衰退，主力可能撤退
4 RSI超买 技术 RSI>80（严重超买）+20
RSI>70（超买）+10 		0 ~ +20 RSI严重超买，短期回调概率大
5 布林带上轨 技术 BB位置>0.9（触及上轨）+15
BB位置>0.8（接近上轨）+8 		0 ~ +15 价格触及布林带上轨，强压力位，回调概率高
6 量价背离 量价 放量下跌（量比>1.5）+20
跌幅>3% + 量比>1.2 +12 		0 ~ +20 放量下跌是主力出货信号，需警惕
7 换手率异常 风险 换手率>20% + 盈利中 +15
换手率>15% + 盈利>5% +8 		0 ~ +15 高换手率+盈利状态，筹码松动，主力可能借机出货
8 MACD死叉 技术 MACD<-0.05 +8 0 ~ +8 MACD死叉区域，趋势转弱
9 涨停次日风险 风险 涨幅>9.5%（接近涨停）+10 0 ~ +10 涨停板次日高开低走概率高，持仓需谨慎
10 T+1限制 规则 T+1未到 -5（降低卖出得分） -5 ~ 0 当日买入不可卖出，T+1规则保护，降低卖出紧迫性
⚠️ 卖出信号说明：
• 卖出信号与买入信号独立计算，专门针对持仓股票
• 止损因子权重最高（+45），优先保护本金
• 止盈因子次之（+40），锁定利润
• 技术面因子辅助判断（RSI/布林带/MACD）
• T+1限制期间卖出得分降低，但不影响止损判断
• 所有信号仅供参考，不构成投资建议
