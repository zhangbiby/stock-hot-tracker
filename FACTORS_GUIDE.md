### 4.1 人气趋势因子

```python
def calculate_rank_trend(code, current_rank, minutes=5):
    """
    计算排名趋势
    排名上升 = 正向
    排名下降 = 负向
    """
    prev_rank = get_prev_rank(code, minutes)
    if prev_rank is None:
        return 0, "排名数据不足"
    
    rank_change = prev_rank - current_rank  # 上升为正
    
    if rank_change >= 10:
        return 20, f"5分钟上升{rank_change}位，人气急升"
    elif rank_change >= 5:
        return 15, f"5分钟上升{rank_change}位，人气上升"
    elif rank_change <= -10:
        return -15, f"5分钟下降{abs(rank_change)}位，人气急降"
    elif rank_change <= -5:
        return -10, f"5分钟下降{abs(rank_change)}位，人气下降"
    else:
        return 0, "排名稳定"
```

### 4.2 量价配合因子

```python
def calculate_volume_price(stock):
    """
    量价配合分析
    量增价涨 = 正向
    量增价跌 = 负向
    """
    change_pct = stock.get('change_pct', 0)
    volume_ratio = stock.get('volume_ratio', 1)
    
    if change_pct > 0 and volume_ratio > 1.5:
        return 15, f"量价齐升，量比{volume_ratio:.1f}"
    elif change_pct > 0 and volume_ratio > 1:
        return 10, f"量价正常，量比{volume_ratio:.1f}"
    elif change_pct < 0 and volume_ratio > 1.5:
        return -15, f"放量下跌，量比{volume_ratio:.1f}"
    else:
        return 0, f"量价正常，量比{volume_ratio:.1f}"
```

### 4.3 技术指标因子

```python
def calculate_technical_score(stock):
    """
    综合技术指标评分
    """
    score = 0
    reasons = []
    
    # RSI
    rsi = stock.get('rsi', 50)
    if rsi < 30:
        score += 10
        reasons.append(f"RSI超卖({rsi:.1f})")
    elif rsi > 70:
        score -= 10
        reasons.append(f"RSI超买({rsi:.1f})")
    
    # MACD
    macd = stock.get('macd', 0)
    if macd > 0:
        score += 5
        reasons.append("MACD正值")
    else:
        score -= 5
        reasons.append("MACD负值")
    
    # 布林带位置
    bb = stock.get('bb_position', 0.5)
    if bb < 0.2:
        score += 5
        reasons.append("接近布林带下轨")
    elif bb > 0.8:
        score -= 5
        reasons.append("接近布林带上轨")
    
    return score, reasons
```

## 📊 五、综合评分计算

### 5.1 评分公式

```
综合得分 = Σ(因子得分 × 因子权重) + ML得分 × 0.15

其中:
- 因子得分: 各因子的原始得分
- 因子权重: 根据市场状态动态调整
- ML得分: 机器学习模型预测得分 (-20 ~ +20)
```

### 5.2 市场状态权重调整

| 市场状态 | 人气权重 | 量价权重 | 技术权重 | 风险权重 |

| 强牛市 | 25% | 25% | 15% | 10% |
| 牛市 | 20% | 20% | 20% | 15% |
| 震荡市 | 20% | 15% | 25% | 20% |
| 熊市 | 15% | 15% | 20% | 30% |
| 强熊市 | 10% | 10% | 15% | 45% |

### 5.3 评分示例

```
股票: 600396 华电辽能

因子得分:
- 人气趋势: +15 (排名上升8位)
- 量价配合: +15 (量增价涨)
- RSI: +10 (RSI=28超卖)
- MACD: +10 (金叉)
- 板块效应: +10 (电力板块热度)

计算:
基础分 = 15×0.2 + 15×0.2 + 10×0.15 + 10×0.15 + 10×0.15 - 5×0.15
      = 3 + 3 + 1.5 + 1.5 + 1.5 - 0.75
      = 9.75

ML分 = 8 × 0.15 = 1.2

综合得分 = 9.75 + 1.2 = 10.95 ≈ 11分

```
