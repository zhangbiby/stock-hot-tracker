# 按钮显示说明

## 为什么看不到买入/卖出按钮？

### 买入按钮

**位置**: "🔥 今日买入机会"面板中

**显示条件**: 有股票触发 `Strong Buy` 或 `Buy` 信号

**当前状态**:
```
信号分布:
- Risk: 67只 (风险信号)
- Caution: 33只 (谨慎信号)
- Strong Buy: 0只 ❌
- Buy: 0只 ❌
```

**原因**: 当前市场整体偏弱，大多数股票风险评分较高，未达到买入阈值。

---

### 卖出按钮

**位置**: 持仓卡片中（红色/橙色边框的卡片）

**显示条件**: 
1. 持仓存在
2. `can_sell = True`（T+1规则，买入次日才能卖出）

**当前状态**:
```
持仓: 1只
- 000001 Test1
  状态: T+1限制
  可卖日期: 3天后(03-30)
  can_sell: False ❌
```

**原因**: 持仓刚买入，受T+1规则限制，暂不可卖出。

---

## 如何触发按钮显示？

### 触发买入按钮

1. **等待市场好转**: 当股票综合得分 ≥40 时会出现 Buy 信号
2. **调整阈值**: 可以临时降低买入阈值（不推荐）

### 触发卖出按钮

1. **等待T+1**: 次日（或T+1后）自动解锁
2. **添加旧持仓**: 添加买入日期为今天之前的持仓
