## 🚀 使用流程

### 买入流程
1. 查看"🔥 今日买入机会"面板
2. 点击心仪股票的"买入"按钮
3. 表单自动填充代码、名称、价格
4. 修改数量（默认100股）
5. 点击"Add"添加持仓

### 卖出流程
1. 查看Portfolio区域
2. 找到红色/橙色标记的持仓
3. 点击"💰卖出"按钮
4. 输入卖出数量
5. 确认卖出

### 查看详情
1. 点击买入面板的"详情"按钮
2. 或点击股票行的"📊"按钮
3. 查看因子分解和评分
