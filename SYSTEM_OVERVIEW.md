### 1.2 信号引擎层

#### 传统信号引擎 (SignalEngine)
**权重分配**:
```
人气趋势:     20%  (排名变化)
量价配合:     20%  (成交量与价格关系)
技术指标:     20%  (RSI/MACD/布林带)
换手率风险:   15%  (过高换手率预警)
板块效应:     15%  (行业热度)
情绪指标:     10%  (新闻情绪)
```

**信号分级**:
| 信号 | 分数范围 | 建议 |
|------|----------|------|
| Strong Buy | ≥70 | 强烈买入 |
| Buy | 55-69 | 建议买入 |
| Hold | 40-54 | 持有观望 |
| Caution | 25-39 | 谨慎对待 |
| Risk | <25 | 风险回避 |

#### 动态信号引擎 (DynamicSignalEngine)
**市场状态自适应**:
| 市场状态 | 特征 | 权重调整 |
|----------|------|----------|
| 强牛市 | 趋势>80, 波动<30 | 增加量价权重 |
| 牛市 | 趋势>60, 波动<40 | 标准权重 |
| 震荡 | 趋势40-60 | 增加技术权重 |
| 熊市 | 趋势<40, 波动>50 | 增加风险权重 |
| 强熊市 | 趋势<20, 波动>60 | 风险权重50% |

#### 多智能体系统 (Multi-Agent)
**智能体角色**:
| 角色 | 职责 | 输出 |
|------|------|------|
| TechnicalAnalyst | 技术分析 | 趋势/支撑/压力 |

**决策流程**:


## 🎯 二、买入卖出判断因子

### 2.1 买入因子 (正向)

| 因子 | 权重 | 计算方法 | 触发条件 |
|------|------|----------|----------|
| 人气上升 | 20% | 排名变化 | 5分钟上升>5位 |
| 量价齐升 | 20% | 成交量+价格 | 量增价涨 |
| RSI低位 | 15% | RSI(14) | RSI<30超卖 |
| MACD金叉 | 15% | MACD | DIF上穿DEA |
| 板块热度 | 15% | 行业涨幅 | 行业Top3 |
| 新闻利好 | 10% | 情感分析 | 正面新闻>3条 |
| ML高概率 | 辅助 | 上涨概率 | >60% |

**Strong Buy条件**:
- 综合得分 ≥70
- 至少4个因子正向
- ML上涨概率 >55%
- 无重大风险因子

### 2.2 卖出因子 (负向)

| 因子 | 权重 | 计算方法 | 触发条件 |
|------|------|----------|----------|
| 人气下降 | 20% | 排名变化 | 5分钟下降>10位 |
| 量价背离 | 20% | 成交量+价格 | 量增价跌 |
| RSI高位 | 15% | RSI(14) | RSI>70超买 |
| MACD死叉 | 15% | MACD | DIF下穿DEA |
| 换手率过高 | 20% | 换手率 | >30% |
| 止盈信号 | 10% | 持仓盈亏 | 盈利>5% |
| ML低概率 | 辅助 | 上涨概率 | <45% |

**卖出分级**:
| 等级 | 得分 | 动作 | 样式 |
|------|------|------|------|
| 强力卖出 | ≥60 | 立即卖出 | 🔴红色脉冲 |
| 建议卖出 | 40-59 | 考虑减仓 | 🟠橙色边框 |
| 持有观察 | 20-39 | 继续观察 | 默认 |
| 继续持有 | <20 | 安心持有 | 默认 |

### 2.3 风险因子 (一票否决)

| 风险类型 | 触发条件 | 处理 |
|----------|----------|------|
| 超高换手 | >40% | 信号降级 |
| 连续涨停 | 3日涨停 | 风险警告 |
| 业绩暴雷 | 新闻负面 | 回避 |
| 大盘暴跌 | 上证<-3% | 暂停买入 |
| T+1限制 | 当日买入 | 禁止卖出 |

## 📝 七、使用流程

### 7.1 日常操作流程

1. **查看买入机会**
   - 打开页面，查看"🔥 今日买入机会"
   - 关注Top 5买入信号
   - 点击"详情"查看因子分析

2. **买入操作**
   - 点击"买入"按钮
   - 确认数量（默认100股）
   - 点击"Add"添加持仓

3. **监控持仓**
   - 查看Portfolio区域
   - 红色脉冲 = 立即卖出
   - 橙色边框 = 建议卖出
   - 点击"💰卖出"执行卖出

4. **查看报告**
   - 点击"生成日报"
   - 查看详细分析报告
   - 可分享至微信

### 7.2 系统维护

1. **数据备份**
   ```bash
   # 备份数据库
   copy output/stock_tracker.db backup/
   ```

2. **模型重训练**
   ```bash
   # 积累足够数据后
   python build_training_data.py
   python train_model.py
   ```

3. **服务重启**
   ```bash
   # 重启Web服务器
   python server.py
   ```
